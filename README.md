### F1 2021 Rich Presence for Discord
Simple Discord RPC client for the game F1 2021.

##### To-do List

**Remember to set the Client ID of your RPC inside index.js and to set the UDP IP to 127.0.0.1!**

##### Test the project
`npm start`

##### Build
`npm run package-win`

Current Version: 1.0.0
